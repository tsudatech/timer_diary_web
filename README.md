- `index.html` - メインの HTML ファイル
- `styles.css` - スタイルシート

## GitHub Pages の設定

1. GitHub リポジトリの設定で「Pages」を有効にする
2. ソースを「Deploy from a branch」に設定
3. ブランチを「main」、フォルダを「/docs」に設定
